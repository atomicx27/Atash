## 🕯️ The Vision
**Atash** (meaning *Fire* in Avestan) is designed to help preserve and celebrate Parsi heritage by connecting individuals through shared traditions, values, and lifestyle preferences. In an increasingly digital world, Atash provides a private, secure, and aesthetically beautiful space for meaningful connections within the community.

## ✨ Premium Features

- **Immersive Discovery**: A high-end, swipe-based interface featuring glassmorphic design and smooth motion transitions.
- **Deep Compatibility**: Beyond just photos, Atash uses a proprietary quiz system to match you based on lifestyle, values, and community involvement.
- **Voucher-Based Privacy**: To maintain the integrity of our private community, new members join via an invite-only voucher system.
- **Real-Time Sanctuary**: Secure, real-time messaging with your matches to foster authentic conversations.
- **Responsive Mastery**: A bespoke UI that feels like a native app on mobile and a gallery-style experience on desktop.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion/React](https://motion.dev/) (Framer Motion)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Design Principles**: Glassmorphism, Editorial Typography (Cormorant Garamond), and Ivories & Maroons palette.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- A Supabase Project

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/atash.git
   cd atash
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env.local` file in the root directory and add your keys (see `.env.example`):
   ```env
   GEMINI_API_KEY="your_api_key"
   NEXT_PUBLIC_SUPABASE_URL="your_supabase_url"
   NEXT_PUBLIC_SUPABASE_ANON_KEY="your_supabase_anon_key"
   ```

4. **Run Locally**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the app.

## 📱 Design Philosophy
Atash utilizes a **"Mobile-First, Desktop-Elegant"** approach. On mobile devices, the app fills the viewport for an immersive native feel. On desktop, it centers the experience within a refined frame, treating the interface as a digital artifact.
